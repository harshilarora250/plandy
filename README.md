# Plandy - Study Planner

Plandy is a lightweight, offline(if hosted by yourself) study planner web app with tasks, calendar view, pomodoro, streak tracking, and a Pomodoro timer. It runs using local storage (no back-end required)

## Features

--Task Management (add, complete, delete)
--Priority system (High / Medium / Low)
--Daily streak tracking
--Pomodoro timer (focus, short, long break)
--Monthly Calendar view with task counts
--Progress dashboard
--Dark / light mode toggle
--Local storage persistance (no login required)

## Tech Stack

--JavaScript

## How to run

### Option 1 (Easy)
Just go on https://plandy.vercel.app/

### Option 2 
Simply curl the github repo, and then use VS Code extension: Live Server or npx serve command in the project terminal, or python -m http.server then run localhost:8000 in your browser.
